# FreedomLife - Инструкция по запуску

## Проблема с npm на Windows

Похоже, есть проблема с npm на вашем компьютере. Вот 3 способа решить:

### Способ 1: Переустановить Node.js

1. Скачайте Node.js с https://nodejs.org/
2. Установите (выберите "Repair")
3. Перезагрузите компьютер
4. Попробуйте снова: `npm run build`

### Способ 2: Использовать Netlify (самый простой)

1. Создайте аккаунт на https://app.netlify.com
2. Перетащите папку `freedomlife-react` в браузер
3. Готово! Сайт будет доступен по ссылке

### Способ 3: GitHub + Vercel

1. Создайте репозиторий на GitHub
2. Загрузите код туда
3. Подключите к Vercel
